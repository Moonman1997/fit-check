# Fit Check — Tester Guide

## What is Fit Check?
A Chrome extension that helps you understand how garments will fit your body using published measurements and your body dimensions.

## Installation
1. Unzip the attached file
2. Open Chrome and go to chrome://extensions
3. Toggle on "Developer mode" in the top right corner
4. Click "Load unpacked" and select the unzipped folder
5. You should see a tape measure icon in your toolbar

## Setup
1. Click the Fit Check icon in your toolbar
2. Click "My Profile"
3. Enter your body measurements in inches
4. Click "Save Measurements"

## How to Use
1. Navigate to any menswear product page
2. Open the size chart on the page (make sure it's visible on screen)
3. Click the Fit Check icon
4. Click "Analyze This Page"
5. The side panel opens with your fit scorecard
6. You can switch between sizes to compare

## Reading the Scorecard
- Each measurement shows a description of how the garment will fit
- The spectrum bar shows where the fit falls on a range
- Hover over any section of the spectrum to see category names
- Reference points like "Your wrist" or "Your waistband" show key body landmarks
- Click "More" to see additional detail
- Click the info icon for a deeper explanation of what the measurement means

## Notes
- Only works on desktop Chrome
- One analysis per minute (API rate limit)
- Works best with brands that publish garment measurement charts
- If a brand only has body measurements, the extension will note this
- Jeans with waist × length sizing are supported but waist and inseam values are approximate
